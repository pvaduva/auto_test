TC1: Test config drive for wrl image
----------------------------------------------------------------
 
Setup: As admin
* write something in my-user-data.txt and my-user-file.txt files
* Boot VM using:
nova boot --key_name=controller-0 --flavor=103 --nic net-id=<public-net0> or <tenant-mgmt-net,vif-model=avp --nic net-id=<internal-net0> or <tenant-net0>,vif-model=avp --config-drive true --user-data ./my-user-data.txt --file /tmp/foobar=./my-user-file.txt --image=wrl5 wrl5-avp-0
* ssh into VM

Verify:
* run df, check /media/hdd mounted on instance (may be hdb, hdc...)
* cat openstack/latest/user_data gives same output as my-user-data.txt
* cat openstack/content/0000 gives same output as my-user-file.txt
